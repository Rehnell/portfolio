Sandwich Bar

The Sandwich Bar is a software program I created using Python. The purpose is for employees to make orders before their lunch break to save staff being in a cue. The software program will let the staff member select what they want for lunch by selecting a sandwich, wrap, crisps and type of drink. The software program then calculates the total amount you need to pay.
